# Agentic Support

This document provides information about the agentic support features available in the AGENTIC framework. Agentic support allows agents to perform tasks autonomously while still being able to interact with users and other agents effectively.

## Features

The agentic support features include:
- **Skill Definition**: Agents can define and utilize specific skills to perform tasks efficiently.
- **Tool Usage**: Agents can use various tools to enhance their capabilities and perform complex tasks.
- **Memory Management**: Agents can manage their memory to retain important information and improve their performance over time.


## Supported Ecosystem
The AGENTIC framework supports a wide range of ecosystems, including:
- **Antigravity**: Support for Antigravity's tools and services to enhance agent performance.
- **Claude**: Integration with Anthropic's Claude language model for advanced conversational abilities.
- **Cursor Support**: Support for custom ecosystems that can be tailored to specific use cases
- **Kiro**: Integration with Kiro for advanced task management and execution capabilities.
- **OpenAI**: Integration with OpenAI's language models for enhanced natural language processing capabilities.

## Claude


The AGENTIC framework provides support for Anthropic's Claude language model, allowing agents to leverage its advanced conversational abilities. This integration enables agents to understand and respond to user inputs more effectively, making interactions more natural and engaging.

Claude configuration can be customized to suit specific use cases, allowing agents to optimize their performance based on the requirements of the task at hand. This support enhances the overall user experience and enables agents to handle a wider range of scenarios with greater efficiency.




## Antigravity

The AGENTIC framework supports Antigravity's tools and services, allowing agents to enhance their performance and capabilities. This integration enables agents to utilize Antigravity's resources for various tasks, improving their efficiency and effectiveness in handling complex scenarios.
