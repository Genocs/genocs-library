# Saga Writer

## When to use this skill
Use this skill when the user needs to manage distributed transactions using sagas...

## Library and tools
1. Write saga using NServiceBus library
2. Use MongoDB to store saga state and data
3. Use Mermaid state diagrams to visualize the saga flow
4. Use Reqnroll specs tests to validate the saga logic

## How to create a saga
1. Define the saga steps and compensating actions

## How to handle failures
1. Implement compensating actions for each step in the saga...
2. Use the saga writer to execute the saga and handle any failures that occur during execution.
