# Genocs.Saga

![Genocs Library Banner](https://raw.githubusercontent.com/Genocs/genocs-library/main/assets/genocs-library-banner.png)

Saga pattern abstractions for distributed workflow orchestration. Supports `net10.0`, `net9.0`, and `net8.0`.

## Installation

```bash
dotnet add package Genocs.Saga
```

## Getting Started

Use this package to define saga workflows and register saga orchestration services in your application.

## Main Entry Points

- `AddSaga`
