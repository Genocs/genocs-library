# Genocs.LoadBalancing.Fabio

![Genocs Library Banner](https://raw.githubusercontent.com/Genocs/genocs-library/main/assets/genocs-library-banner.png)

Fabio load-balancing integration for service-to-service calls. Supports `net10.0`, `net9.0`, and `net8.0`.

## Installation

```bash
dotnet add package Genocs.LoadBalancing.Fabio
```

## Getting Started

Use this package to register Fabio support in your Genocs builder and create named HTTP clients for Fabio-routed services.

## Main Entry Points

- `AddFabio`
- `AddFabioHttpClient`
