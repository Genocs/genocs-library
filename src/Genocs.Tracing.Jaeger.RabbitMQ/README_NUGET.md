# Genocs.Tracing.Jaeger.RabbitMQ

![Genocs Library Banner](https://raw.githubusercontent.com/Genocs/genocs-library/main/assets/genocs-library-banner.png)

Jaeger tracing integration for RabbitMQ-based messaging pipelines. Supports `net10.0`, `net9.0`, and `net8.0`.

## Installation

```bash
dotnet add package Genocs.Tracing.Jaeger.RabbitMQ
```

## Getting Started

Use this package to add Jaeger trace propagation and instrumentation for RabbitMQ messaging flows.

## Main Entry Points

- `AddJaegerRabbitMqPlugin`
