# RabbitMQ Message Broker for Genocs Message Brokers

[![NuGet](https://img.shields.io/nuget/v/Genocs.MessageBrokers.RabbitMQ.svg)](https://www.nuget.org/packages/Genocs.MessageBrokers.RabbitMQ/)
[![NuGet](https://img.shields.io/nuget/dt/Genocs.MessageBrokers.RabbitMQ.svg)](https://www.nuget.org/packages/Genocs.MessageBrokers.RabbitMQ/)

This package contains RabbitMQ message broker integration functionalities designed by Genocs.
The library is built to be used with .NET10, .NET9, .NET8.

## Description

RabbitMQ Message Broker NuGet package provides message queuing, publish/subscribe patterns, and reliable message delivery for distributed systems and microservices architectures.
