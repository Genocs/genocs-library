# The Genocs Library - Distributed Tracing components

[![NuGet](https://img.shields.io/nuget/v/Genocs.Tracing.svg)](https://www.nuget.org/packages/Genocs.Tracing/)
[![NuGet](https://img.shields.io/nuget/dt/Genocs.Tracing.svg)](https://www.nuget.org/packages/Genocs.Tracing/)

This package contains distributed tracing functionalities designed by Genocs.
The library is built to be used with .NET10, .NET9, .NET8.

## Description

Tracing NuGet package provides distributed tracing capabilities using OpenTelemetry and Jaeger for monitoring and troubleshooting microservices.

Please check the [GitHub repository](https://github.com/Genocs/genocs-library) to get more info.
