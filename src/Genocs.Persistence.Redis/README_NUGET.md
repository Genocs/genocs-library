# Genocs.Persistence.Redis

![Genocs Library Banner](https://raw.githubusercontent.com/Genocs/genocs-library/main/assets/genocs-library-banner.png)

Redis-backed persistence helpers and repository integration. Supports `net10.0`, `net9.0`, and `net8.0`.

## Installation

```bash
dotnet add package Genocs.Persistence.Redis
```

## Getting Started

Use this package to integrate Redis-backed persistence components in Genocs applications.

## Main Entry Points

- `AddRedis`
