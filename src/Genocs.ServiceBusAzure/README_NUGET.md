# The Genocs Library - Azure Service Bus Message Broker components

## Description
**Genocs.MessageBrokers.AzureServiceBus** provides Azure Service Bus integration for reliable message queuing, pub/sub patterns, and enterprise messaging scenarios targeting .NET10, .NET9, .NET8.

## Dependencies
- **Genocs.Core** (5.0.9)
- **Genocs.MessageBrokers** (5.0.9)
- **Azure.Messaging.ServiceBus** (7.18.1)
- **Microsoft.Extensions.Azure** (1.7.5)

### Framework references
- Microsoft.NETCore.App
